# Project 2 - Signup Form

## Description
A professional and elegant signup form built with HTML5 and CSS3, matching the provided Figma design specifications. This project features a dark-themed interface with clean typography and proper form validation.

## Features
- ✅ Complete HTML5 structure with semantic tags
- ✅ All necessary head tags (title, meta, favicon, description)
- ✅ Proper form validation with `required` attribute
- ✅ Dark theme matching Figma design
- ✅ Responsive design for all devices
- ✅ Organized sections (header, main, footer)
- ✅ Input fields for:
  - Full Name
  - Email
  - Password
  - Confirm Password
  - Date of Birth (MM/DD/YYYY format)
  - User Type (Student/Employee radio buttons)
  - Terms and Conditions checkbox
- ✅ Submit button with hover effects

## Technologies Used
- HTML5
- CSS3
- Libre Baskerville font (Google Fonts compatible)

## File Structure
```
Project 2-Signup/
├── index.html      # Main HTML file with form structure
├── styles.css      # Complete styling matching Figma design
└── README.md       # Project documentation
```

## How to Run
1. Open `index.html` in any modern web browser
2. The form will be displayed with all styling applied
3. Fill out the form fields and click "Signup" to test validation

## Design Specifications
- **Background**: Black (#000000)
- **Text Color**: White (#FFFFFF)
- **Font**: Libre Baskerville (serif)
- **Form Title Size**: 64px
- **Input Fields**: Bottom border style with transparent background
- **Button**: White background with black text

## Form Validation
All input fields are marked as `required` to ensure:
- Users cannot submit empty forms
- Email field validates proper email format
- Password fields require minimum 6 characters
- Date of birth follows MM/DD/YYYY pattern
- User type must be selected
- Terms and conditions must be accepted
